# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-16

### Added
- Initial release of react-paginated-multiselect
- Multi-select support with checkbox selection
- Infinite scroll with auto-fetch on scroll
- Server-side pagination support
- Async search with debounced API calling
- Virtual scrolling using react-window
- Fully customizable UI without Material-UI dependency
- TypeScript support with full type definitions
- Default CSS styles with customization options
- Select All functionality
- Chip display for selected items
- Clear button functionality
- Custom option rendering support
- Loading states and spinners
- Search with multiple terms (semicolon-separated)
- Comprehensive documentation and examples

### Features
- Zero UI framework dependencies (no MUI)
- Works with any backend API
- Handles large datasets efficiently (10,000+ items)
- Fully accessible with ARIA attributes
- Mobile-friendly and responsive
- Supports both single and multi-select modes
