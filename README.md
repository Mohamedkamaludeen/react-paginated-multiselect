A reusable React multi-select dropdown component featuring infinite scrolling, server-side pagination, and real-time search support.
This package allows developers to efficiently load large datasets from APIs using paginated fetch calls triggered on scroll, while also supporting type-ahead searching using async server queries. Ideal for applications that handle thousands of records without sacrificing performance.
